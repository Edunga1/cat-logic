2.6.12л¶Җн„° SETNX Deprecated лҗҳкі , setм—җ NX мҳөм…ҳмқҙ м¶”к°Җлҗҳм—ҲлӢӨ.

SETNX is short for "SET if Not eXists".

NXлҠ” Not eXistsмқҳ м•ҪмһҗлЎң, н•ҙлӢ№ нӮӨк°Җ мЎҙмһ¬н•ҳм§Җ м•Ҡмқ„ л•Ңл§Ң SETмқ„ мҲҳн–үн•ңлӢӨ.

The command SET resource-name anystring NX EX max-lock-time is a simple way to implement a locking system with Redis.

SET resource-name anystring NX EX max-lock-timeмҷҖ к°ҷмқҙ мӮ¬мҡ©н•ҳлҠ” кІғмқҖ Redisм—җм„ң Locking Systemмқ„ кө¬нҳ„н•ҳлҠ” к°„лӢЁн•ң л°©лІ•мқҙлӢӨ.

EX max-lock-timeмңјлЎң мқён•ҙ мҙҲ лӢЁмң„лЎң л§ҢлЈҢ мӢңк°„мқ„ м§Җм •н•  мҲҳ мһҲлӢӨ. мһҳ м„Өм •лҗҳм—ҲлӢӨл©ҙ "OK"лҘј л°ҳнҷҳн•ҳкі , л§ҢлЈҢ мӢңк°„мқҙ м§ҖлӮҳкё° м „м—җ лӢӨмӢң нҳём¶ңн•ҳл©ҙ nilмқ„ л°ҳнҷҳн•ңлӢӨ.

мқҙ л°©мӢқмңјлЎң к°„лӢЁн•ң л¶„мӮ° Locking Systemмқ„ кө¬нҳ„н•  мҲҳ мһҲлӢӨ.

нҒҙлқјмқҙм–ёнҠём—җм„ң к°Ғмһҗ нҸ¬мқёнҠёлҘј м ҒлҰҪ, м°Ёк°җн•ҳлҠ” лЎңм§Ғмқҙ мһҲлӢӨкі  к°Җм •н•ҳмһҗ.

Process:

  • к°Ғ нҒҙлқјмқҙм–ёлҠ” мң„ лӘ…л №м–ҙлЎң нҸ¬мқёнҠёлҘј мІҳлҰ¬н•ҳкё° м „м—җ Locking н•ңлӢӨ: SET point-user123 foo NX EX 60
  • л§Ңм•Ҫ Lockingм—җ мӢӨнҢЁн•ҳл©ҙ(nilмқ„ л°ҳнҷҳн•ҳл©ҙ) "лӢӨлҘё нҒҙлқјмқҙм–ёнҠём—җм„ң мІҳлҰ¬ мӨ‘мһ…лӢҲлӢӨ."мҷҖ к°ҷмқҖ л©”мӢңм§ҖлҘј л°ҳнҷҳн•ңлӢӨ.
  • Lockingм—җ м„ұкіөн•ҳл©ҙ("OK"лҘј л°ҳнҷҳн•ҳл©ҙ) нҸ¬мқёнҠёлҘј мІҳлҰ¬н•ңлӢӨ.
  • мІҳлҰ¬к°Җ лҒқлӮҳл©ҙ DEL point-user123лЎң Lockingмқ„ н•ҙм ңн•ңлӢӨ.

м„Өл № нҒҙлқјмқҙм–ёнҠёк°Җ мІҳлҰ¬ мӨ‘м—җ л№„м •мғҒ мў…лЈҢлҗҳлҚ”лқјлҸ„ 60мҙҲ нӣ„м—җлҠ” Lockingмқҙ н•ҙм ңлҗҳкё° л•Ңл¬ём—җ л¬ҙн•ңм • Locking лҗҳлҠ” кІғмқ„ н”јн•  мҲҳ мһҲлӢӨ.

мўҖ лҚ” кІ¬кі н•ҳкІҢ Unlocking н”„лЎңм„ёмҠӨлҘј л§Ңл“Өкі  мӢ¶лӢӨл©ҙ DEL лӘ…л №м–ҙлҘј м§Ғм ‘ мӮ¬мҡ©н•ҳм§Җ м•Ҡкі , мқјл Ёмқҳ н•ҙм ң н”„лЎңм„ёмҠӨлҘј нҳём¶ңн•ҳлҠ” кІғмқҙ мўӢлӢӨ:

  • fooмҷҖ к°ҷмқҖ кі м • л¬ёмһҗм—ҙ лҢҖмӢ , лһңлҚӨн•ң л¬ёмһҗм—ҙмқ„ мӮ¬мҡ©н•ңлӢӨ.
  • нҒҙлқјмқҙм–ёнҠёмқҖ нҸ¬мқёнҠё мІҳлҰ¬ нӣ„ н•ҙм ң н”„лЎңм„ёмҠӨм—җ мң„ л¬ёмһҗм—ҙмқ„ м „лӢ¬н•ңлӢӨ.
  • н•ҙм ң н”„лЎңм„ёмҠӨлҠ” м „лӢ¬ л°ӣмқҖ л¬ёмһҗм—ҙкіј GET лӘ…л №м–ҙлЎң н•ҙлӢ№ нӮӨмқҳ к°’мқ„ л№„көҗн•ңлӢӨ.
  • к°ҷмңјл©ҙ DEL лӘ…л №м–ҙлЎң Lockingмқ„ н•ҙм ңн•ңлӢӨ.

к°Ғ нҒҙлқјмқҙм–ёнҠём—җм„ң м§Ғм ‘ мӮӯм ңн•ҳлҠ” лҢҖмӢ , н•ҙм ң н”„лЎңм„ёмҠӨм—җ мң„мһ„н•ҳлҠ” л°©лІ•мқҙлӢӨ.
мқҙлҹ¬л©ҙ лӢӨлҘё нҒҙлқјмқҙм–ёнҠёк°Җ DEL лӘ…л №м–ҙлЎң м•„л¬ҙ л°°кІҪм—Ҷмқҙ Locking н•ҙм ңн•ҳлҠ” кІғмқ„ л§үмқ„ мҲҳ мһҲлӢӨ. Locking мІҳлҰ¬н•ң нҒҙлқјмқҙм–ёнҠёл§Ң н•ҙм ңн•  мҲҳ мһҲлҠ” мқјмў…мқҳ ліҙмҰқ м Ҳм°ЁлҘј л§Ңл“ңлҠ” кІғмқҙлӢӨ.

лқјмқҙм„ мҠӨлҘј BSDм—җм„ң SSPLлЎң ліҖкІҪ

24л…„ 4мӣ”, нҒҙлқјмҡ°л“ң м„ңл№„мҠӨ лҢҖн•ӯкІ©мқё лқјмқҙм„ мҠӨ ліҖкІҪмқ„ л°ңн‘ңн–ҲлӢӨ.

л¬ём ңк°Җ лҗҳлҠ” SSPL лқјмқҙм„ мҠӨлҠ” MongoDBк°Җ мІҳмқҢмңјлЎң лҸ„мһ…н–Ҳкі , мқјл°ҳ мӮ¬мҡ©мһҗм—җкІҢлҠ” мҳҒн–Ҙмқҙ м—Ҷм§Җл§Ң, нҒҙлқјмҡ°л“ң м„ңл№„мҠӨ м—…мІҙм—җлҠ” м ңм•Ҫмқҙ мғқкёҙлӢӨкі  н•ңлӢӨ.

мқҙлЎң мқён•ҙ RedisлҘј нҳёнҷҳн•ҳлҠ” н”„лЎңм қнҠёл“Өмқҙ мғқм„ұлҗҳкі  мһҲлӢӨ.
