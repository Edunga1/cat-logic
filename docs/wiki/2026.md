# 2026년

요즘 사용하는 AI 도구
- [Gemini](/docs/wiki/google-gemini.md)는 모델 업데이트가 정체되어 사실상 사용하지 않는다.
- [Copilot](/docs/wiki/github.md#github-copilot)의 자동 완성을 주로, CLI 에이전트 코딩은 주로 Claude Code를 사용한다. 그래도 구독은 유지중.
- [Claude Code](/docs/wiki/anthropic.md#claude-code)
    - MCP, Skills, Subagent는 구현하고 사용해 보았다. 결국 다 비슷한 개념이지만.
    - Teammates는 다음으로 시도해 볼 개념인데, 지금 업무 처리에는 과도한 느낌이 든다.

OSS 종료된 프로젝트가 눈에 띈다.
[Localstack](/docs/wiki/aws.md#localstack)과 [nvim-treesitter](/docs/wiki/vim.md#구문-강조)는 각각 3월, 4월에 저장소가 아카이브되었다. Localstack은 무료 소프트웨어 라이센스를 폐기했고 nvim-treesitter는 유지보수자의 번아웃 추정.
treesitter는 그래도 neovim으로 코어 편입이 어느정도 되었다.

맥 터미널 에뮬레이터를 iTerm2에서 [WezTerm](/docs/wiki/mac-os.md#터미널에서-한글을-두-번-입력해야-하는-문제)으로 바꿨다.
기본앱, kitty, iTerm2 모두 한글 조합 중에 엔터를 두 번 눌러야 조합완성 후 입력까지 되는 문제가 있는데,
WezTerm은 조합 중에 입력까지 한 번에 되는 유일한 터미널 에뮬레이터였다.
