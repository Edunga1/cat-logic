# 2025년

요즘 사용하는 Agent 도구

- [copilot-cli](/docs/wiki/machine-learning.md#copilot-cli)
- [gemini-cli](/docs/wiki/machine-learning.md#gemini-cli)

copilot-cli는 [GitHub Copilot](/docs/wiki/github.md#github-copilot)을 구독하고 있어서 사용 중이다.
월 300개의 요청을 무료로 사용할 수 있다.
gemini-cli는 무료 계정으로, 2.5-flash 모델로 사용 중이다.
가끔 회사 계정으로 2.5-pro 모델을 사용하는데, 사용한 만큼 청구되어 잘 사용하지 않는다.

이외의 AI 도구는 Claude Desktop + MCP 조합이다.
MCP는 사내 개발용 API를 대신 호출하게 하거나 Jira 티켓을 검색하는 용도로 사용한다.

**사내 개발 API 호출하는 MCP를 만들면서 즐거웠던 점**

Claude Chat에 MCP를 연결하고, AI에게 호출해 보게 하고 피드백 받는 과정이 재밌었다.
수정 사항을 반영한 뒤 같은 대화 세션에서 다시 시도해보라고 하면 피드백 과정이 많이 단축된다.
마치 아이언맨이 자비스를 개선하는 느낌. 비록 수정은 내가 하지만.
2025년 하반기부터는 [MCP를 지원하는](/docs/wiki/machine-learning.md#copilot-cli) [에이전트](/docs/wiki/machine-learning.md#gemini-cli)가 많아져서,
AI가 스스로를 위해 MCP를 개선하거나, 필요한 도구를 만들도록 시킬 수 있을 것 같다.

copilot은 환율 이슈로 매우 부담될 예정.
1월에 갱신되는데, 12월 24일 기준으로 $100가 14만 5천원이다.
